#The WindLog

A windsurfing sessions diary implemented using a MEAN stack:

* Server dependencies:
    - connect-flash
    - cookie-parser
    - express
    - underscore
    - passportjs
    - mongodb
    - mongoose
    - grunt
    - grunt-nodemon
    - vash
    - bower

* Client side dependencies:
    - angular
    - angular-ui-notification
    - jquery
    - angular-smart-table
    - angular-route
    - bootstrap
    - underscore
    - angular-bootstrap
    
___


Database: Mongo.

Authorizing: Passport with local authorization (Username/Password).

View engine: vash.


Install process:

Clone repository:

    `git clone https://github.com/ugeHidalgo/TheWindLog.git`

Install:

    ```cd thewindlog
    npm install
    bower install```
